Intro CSS - BoxModel

Pour un élément box (avec la bordure noir) vous aurez toujours :
  • Les marges extérieurs (en vert) => margin
  • Les marges intérieurs (en orange) => padding
  • la bordure (en noir) => border
  • la taille du contenu (en blanc) => width height
Ces éléments sont visibles et modifiables à la volée dans le panneau du développeur de votre navigateur en bas des propriétés CSS (MAJ+CTRL+I ou F12)

Réalisation simple d'une "cible" en content-box
  • Réalisez le script permettat de créer 4 carrés (divs) de 50px disposant chacune de bordures différentes avec un texte à l'intérieur Bordures
  • Utilisez des marges différentes
    • 1er carré : un espace de 50px en haut et en bas sans padding
    • 2ème carré : margin => 50px 30px 40px et un padding de 20px
    • 3ème carré : margin 50px et 40px puis un padding de 5px 20px 30px
    • le dernier carré : sans marge et un padding de 10px et 20px
