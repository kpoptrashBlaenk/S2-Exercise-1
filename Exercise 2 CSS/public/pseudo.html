  • Lors de l'activation : changer la taille du texte

Exercice : Image en pseudo élément

Réalisez le script permettant de :
  • Créez une boîte (div) contenant l'image d'un produit, un texte explicatif et un bouton achat à la manière d'un site de e-commerce
  • Lors du survol de l'image (ou du bloc contenant l'image), utilisez le CSS et les pseudo-elements pour faire apparaitre un texte indiquant que le produit est en promotion
  • Lors du survol du bouton d'achat, utilisez le CSS pour faire apparaitre une image indiquant les méthodes de paiements
Des images d'exemple sont disponibles dnas le dossier assets/img
